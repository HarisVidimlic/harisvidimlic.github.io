Homemade board game

A super fun project where I designed a board game using Fusion 360 and printed it at home on my 3D printer. Plastic marbles were also 3D printed and used as the game pieces. It grew very popular among friends and family, so I had to start "mass-producing" it as you can see on the buttom right image. A good number of people even wanted to pay for it which has paid for all the filament. 🤑 As of now, I've only written the rule book in Swedish. The board game resembles the game Fia in some ways, but has many unique aspects, and I think - is much more fun.
